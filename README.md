# What is JavaScript?
 JavaScript is a programming language that is commonly used in web development. It is a client-side language, which means that it is executed by the user's web browser rather than the web server. This allows JavaScript to interact with the user and create dynamic, interactive web pages.
 
## “.js” is the extension.
- Nowadays used in server-side development.
- JS Frontend Frameworks: React, Angular, Vue.
- JS Backend Frameworks: Express, Node.

  # How to install Node js ?
## Answer:  Node.js Installation

Node.js Installation
- Node.js is an open-source, cross-platform JavaScript runtime environment that allows JavaScript to run outside a web browser. It is widely used for server-side applications and command-line tools.

## Steps to Install Node.js:
- Visit the official website: https://nodejs.org
- Download the latest Node.js installer.
-Run the installer and follow the on-screen instructions.
-Verify the installation by opening a terminal and typing:
```
node -v
```
The installed Node.js version will be displayed.

## Runing JavaScript Using Node.js in VS Code

- Install Visual Studio Code from https://code.visualstudio.com
-Create a new file with a .js extension and write JavaScript code.
-Open the terminal in VS Code.

# Run the file using the command:
```
     node filename.js
```

### The output will appear in the terminal.
